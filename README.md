## 📌 Overview
This project converts two different telemetry data formats into a single unified structure.

## ⚙️ Features
- Handles two different JSON input formats
- Converts ISO 8601 timestamps to milliseconds (epoch time)
- Transforms nested and flat data into a standardized format
- Includes automated unit tests

## 📂 Files
- `main.py` → Core logic and test cases
- `data-1.json` → Input format 1
- `data-2.json` → Input format 2
- `data-result.json` → Expected unified output

## ▶️ How to Run
```bash
python main.py
